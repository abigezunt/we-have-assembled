About Web Development Immersive

WDI is General Assembly's Web Development Immersive Track. Fourteen students left their jobs and colleges to immerse themselves in learning web development all day, every day, plus homework every evening and weekend for twelve weeks. This is what we've learned.

What we've learned

Ruby on Rails

We started with programming in pure Ruby, and progressed into building web sites with Sinatra and embedded Ruby, incorporating external APIs, and buiding beautiful, responsive web apps with Rails.

JavaScript

We've also learned to build websites on the client side, using JavaScript, jQuery, AJAX, d3.js, and AngularJS to build interactive and fast websites.

Open Source

We built and published Ruby gems to help other Ruby programmers access APIs as our first big contributions to the open source community.

The Full Stack

We've become talented full-stack developers, learning to use PostgreSQL, Heroku, and AWS, caching with Redis, load testing servers, serving static assets from cloud, and TDD with RSpec, Jasmine, Capybara, Webmock, and Factory Girl.

Featured Projects

Only 12 weeks? Impossible.

We didn't just practice coding - we built interesting, beautiful, useful projects using Ruby on Rails and JavaScript. Here is a sampling of the cool things we coded during our 12 weeks at WDI.

LadyParts

Ladyparts is a web app that explores films that depict multifaceted female characters using the Bechdel Test and the Rotten Tomatoes API and presents the dataset using interactive d3.js graphs.

By: Cori Allen and Emma Spencer

Esh Circus Arts

Software to help a local circus school handle class registrations, attendance tracking, credit card payments, and calendar integration.

By: Abby Howell

Monroe

An API that collects tweets and newspaper articles and provides real-time sentiment analysis about politicians.

By: Blake Ruddock, Ben Winter, and Bryce Lambert

Rethinking Home

An app that collects and maps stories on how climate change affects the natural and human environment on a local scale.

By: Emma Spencer

NY Times Bestsellers

A ruby gem for making API calls to the New York Times Bestsellers list.

By: Emma Spencer

RubyKhan

A Ruby gem for accessing the Khan Academy API.

By: Dan Crews and Abby Howell

Notebook Printer

Convert your Evernote notebooks to PDFs using this practical, beautiful Sinatra app.

By: Bryce Lambert

ClimbKeeper

Review indoor rock climbing routes

By: Matt Clement

New England Town Name Generator

Wondering what to name your new settlement? Get some suggestions with this hilarious app.

By: Chris Sherlin

Cat's Cuties

E-commerce for a local, handmade, organic, vegan, artisan chocolate company.

By: Abby Howell

EnviroView

Visualizations of emissions data from the EPA and the Sunlight Foundation API.

By: Dan Crews, Matt Clement, and Abby Howell
